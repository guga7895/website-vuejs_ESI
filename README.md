## Rodar Testes Cucumber

Para rodar os testes cucumber locais (no localhost), utilizar: 
```
cucumber -p localhost
```
Para rodar os testes cucumber no website funcionando (cuidando.vc), utilizar:
```
cucumber -p cuidando_vc
```
